# Rejang-AI

## Description
Rejang AI is an artificial intelligence-based application designed to preserve and introduce Rejang language and culture from Bengkulu Province, Indonesia.

## Features
- History
- Dictionary
- Translation
- Chatbot
- Kaganga Recognition

## Technologies
- PHP
- MySQL
- Bootstrap
- jQuery
- AJAX
- Web Speech API
